## Support CSS Autocomplete

### Summary
Enables the autocomplete feature for CSS.

### Syntax
```vim
'' type this before starting
:set omnifunc=csscomplete#CompleteCSS

'' then launched using
Ctrl+X Ctrl+O

'' to move inside the list use
Ctrl+O '' one element down
Ctrl+P '' one element up
```
