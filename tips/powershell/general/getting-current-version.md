## Getting the currently installed version

### Summary
Allows the developer to obtain the currently installed version of PowerShell

### Syntax
```powershell
$PSVersionTable.PSVersion
```
### Examples
```powershell
$ $PSVersionTable.PSVersion  
  Major  Minor  Build  Revision
  -----  -----  -----  --------
  4      0      -1     -1
```
