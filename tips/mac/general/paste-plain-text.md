## Paste special \(plain text\)

### Summary
Allows the user to paste a plain copy of a rich text, thus removing all formatting.  
  
### Syntax
- Mac OSX
```
Command + Shift + V  
```

- Windows
```
Ctrl + Alt + V  
```
